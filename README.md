# WebArcade

Classic arcade games recreated for the browser using Claude Code.

## Games

- **Bounce** - Keep the ball from falling
- **Brick Bounce** - Break all the bricks
- **Brick Bounce 2** - Advanced brick-breaking with power-ups and levels
- **Brick Spinner** - Stack and clear falling blocks
- **Serpentine** - Grow your snake, avoid yourself
- **Galactic Defense** - Defend Earth from alien invaders

## Features

- Keyboard and gamepad support (Xbox, SNES-style controllers)
- Procedural chiptune sound effects and music
- Local high score persistence
- No server required - runs entirely in browser

## Development

See `CLAUDE.md` for development patterns and the shared module APIs.

### Debug Utility

`debug.html` provides a controller testing page for verifying input mappings and troubleshooting gamepad issues.
