All arcade games support keyboard (Arrow keys / WASD) and gamepad controllers (Xbox, SNES-style).
D-Pad or Left Stick to move, A button to select, Start to pause.

About WebArcade

WebArcade is an experiment in using Claude Code to rapidly recreate classic arcade game experiences. Each game runs entirely in your browser with no server required. All games feature keyboard and gamepad support, sound effects, and local high score persistence.
